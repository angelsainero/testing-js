# Testing JS

Para hacer los tests de integración deberemos lanzar el server. Para ello necesitaremos 2 terminales, uno para el server y otro para la DB

### Instalación

```bash
cd server
npm i
```

### Levantar servicios
#### DB
```bash
cd server
npm run db:up
```

#### Server
```bash
cd server
npm run start:dev
```
